# Privacy Policy / プライバシーポリシー

**GitHub Gearbox**  
Last updated / 最終更新日：2026-02-21

---

## English

### Overview

GitHub Gearbox is a Chrome extension that enhances productivity on GitHub. This Privacy Policy explains what data the extension accesses, how it is used, and your rights as a user.

### Data Collection

**GitHub Gearbox does not collect, transmit, or store any personal data.**

The extension operates entirely within your browser and does not send any information to external servers operated by the developer.

### Data Accessed Locally

The extension accesses the following data solely within your browser to provide its features:

| Data | Purpose |
|------|---------|
| GitHub page content (DOM) | To enable task-list click expansion, review comment copying, and badge insertion |
| Browser local storage (`chrome.storage`) | To save extension settings (e.g., enabled/disabled state of each feature) |

All data access is local only. Nothing is transmitted outside your browser.

### Permissions

The extension requests the following Chrome permission:

| Permission | Reason |
|-----------|--------|
| `storage` | To persist user preferences across browser sessions |

The extension operates only on pages matching `https://github.com/*`.

### Third-Party Services

This extension does not integrate with any third-party analytics, advertising, or tracking services.

### Data Sharing

No data is collected, therefore no data is shared with any third party.

### Changes to This Policy

If this Privacy Policy is updated, the new version will be published in this repository. Continued use of the extension after changes are posted constitutes acceptance of the updated policy.

### Contact

For questions or concerns regarding this Privacy Policy, please open an issue on the [GitHub repository](https://github.com/iririn1998/github-gearbox).

## 日本語

### 概要

GitHub Gearbox は、GitHub 上での生産性を向上させる Chrome 拡張機能です。このプライバシーポリシーでは、本拡張機能が取り扱うデータ、その利用目的、およびユーザーの権利について説明します。

### データの収集

**GitHub Gearbox は、個人データの収集・送信・保存を一切行いません。**

本拡張機能はお使いのブラウザ内でのみ動作し、開発者が管理する外部サーバーへいかなる情報も送信しません。

### ローカルでアクセスするデータ

本拡張機能は、各機能を提供するために以下のデータへブラウザ内でのみアクセスします。

| データ | 目的 |
|--------|------|
| GitHub ページのコンテンツ（DOM） | タスクリストのクリック領域拡大・レビューコメントのコピー・バッジ挿入の実現 |
| ブラウザのローカルストレージ（`chrome.storage`） | 各機能の有効・無効などの設定を保存するため |

すべてのデータアクセスはローカルに限定されます。ブラウザ外部へのデータ送信は行いません。

### 権限（パーミッション）

本拡張機能は以下の Chrome 権限を要求します。

| 権限 | 理由 |
|------|------|
| `storage` | ブラウザセッションをまたいでユーザー設定を永続化するため |

本拡張機能は `https://github.com/*` に一致するページでのみ動作します。

### サードパーティサービス

本拡張機能は、サードパーティの解析・広告・トラッキングサービスとは一切連携していません。

### データの共有

データを収集しないため、第三者へのデータ提供は行いません。

### ポリシーの変更

本プライバシーポリシーを変更する場合は、本リポジトリに新しいバージョンを公開します。変更後も引き続き本拡張機能をご利用いただいた場合、更新後のポリシーに同意したものとみなします。

### お問い合わせ

本プライバシーポリシーに関するご質問やご不明点は、[GitHub リポジトリ](https://github.com/iririn1998/github-gearbox)のIssueよりお知らせください。
